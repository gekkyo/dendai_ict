## 制作環境

#### OS

* MacOS 10.15.7

#### 使用言語

* python = ">=3.9,<3.10"

#### ライブラリ

* chromalog = "^1.0.5"
* PySimpleGUI = "^4.49.0"
* matplotlib = "^3.4.3"
* pathlib = "^1.0.1"
* pandas = "^1.3.3"
* scipy = "^1.7.1"
* pyserial = "^3.5"
* statsmodels = "^0.13.0"
* seaborn = "^0.11.2"

## Install Packages

```
poetry install
```

もしくは

```
pip install -r requirements.txt
```
